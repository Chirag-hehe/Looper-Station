Support

Setup notes for recording and hardware.

The browser version is useful for testing the workflow. For stage-grade performance, test with the exact phone, browser, audio interface, and controller you plan to use.

Mic access

Open the site from localhost or HTTPS, then select Enable mic. If the browser blocks the prompt, check site permissions and reload.

Audio interface

Connect the interface before enabling the mic. After permission is granted, choose the device from the Input menu. Wired headphones reduce feedback and latency.

MIDI controller

Enable MIDI from the control panel. Notes 36-41 or CC 20-25 control tracks 1-6. Note 42 or CC 102 starts playback. Note 43 or CC 103 stops playback.

Saved takes

Saved takes live in the current browser profile. Clearing site data can delete them. Export WAV files for anything you want to keep outside the browser.
