Privacy

Local-first by design.

Looper Station is built so your recordings and saved takes stay in your browser unless you choose to export them. No accounts, no uploads, no tracking.

Data handled

The app can request microphone access, MIDI access, and local browser storage. Audio is processed entirely in the browser for live looping. Saved takes are stored locally using IndexedDB on the device and browser you use.

Data not collected

Looper Station does not send recordings, MIDI messages, saved sessions, or microphone audio to any server. Exported WAV files are created in your browser and downloaded to your device only. There is no analytics, crash reporting, or telemetry.

Permissions

Microphone permission is required to record audio. MIDI permission is optional and only needed for external controllers. Browser storage (IndexedDB) is used for saved takes. The service worker caches app files for offline use.

Hosting

Looper Station is hosted on GitHub Pages. GitHub may collect standard server access logs (IP address, browser, referrer) as part of serving the site. See GitHub's privacy statement for details. No additional analytics are added by this project.

Android app

If a Play Store version is released in the future, this policy will be updated with the package name, support contact, crash reporting choices, and any cloud features added at that time.
